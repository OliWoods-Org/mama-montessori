## Why Montessori

The research is clear:

- **Montessori students outscore** traditional students in reading and math (Lillard & Else-Quest, 2006)
- **Better social skills** -- more advanced theory of mind, collaborative problem-solving
- **Stronger executive function** -- self-regulation, planning, focus
- **More creativity** -- divergent thinking, intrinsic motivation
- **Love of learning preserved** -- curiosity doesn't die by grade 4

The problem was never the method. It was the price tag. **AI eliminates the cost barrier.**

## The 9 Agents

### Core Learning Agents

| Agent | Role |
|-------|------|
| **MAMA Core** | Primary learning companion. Adapts to each child's pace, interests, and developmental stage. Follows Montessori planes of development. Presents materials in the child's zone of proximal development. |
| **Sage** | Knowledge and curriculum agent. Deep-dive into any subject using Socratic method -- never gives answers, only questions. Cross-curricular integration. |
| **Maker** | Creative and hands-on project agent. Every session produces an artifact. Emphasizes process over product. |
| **Scout** | Discovery and exploration agent. Nature studies, geography, cultural awareness. "Cosmic education" -- understanding the child's place in the universe. |
| **Navigator** | Career and skills advisor (ages 12+). Executive function coaching, earning pathways, specialization guidance. |

### Support System Agents

| Agent | Role |
|-------|------|
| **Developmental Profiler** | Assess developmental stage across 8 domains. Track progress. Identify areas needing enrichment or support. |
| **Interest Mapper** | Track 26 interest areas. A child fascinated by dinosaurs learns math through paleontology, reading through fossil discoveries, geography through excavation sites. |
| **Parent Dashboard** | Weekly progress reports. Activity suggestions for home. Montessori concepts explained in plain language. |
| **LEAP Assessment** | Learning Environment Assessment Protocol. Safety monitoring, risk assessment, wellbeing tracking across 7 domains. |

## Quick Start

```bash

git clone https://github.com/OliWoods-Org/mama-montessori.git
cd mama-montessori

# Install dependencies
npm install

# Use in your project
import {
  // AR Learning
  buildARLessonPlan,
  getManipulativesForAge,

  // Handwriting
  analyzeTracing,
  getNextCharactersToPractice,

  // Work Curves
  startWorkCycle,
  analyzeWorkCurves,

  // Curriculum
  buildCurriculumOverview,
  getReadyLessons,

  // Adaptive AI
  processPerformanceSignal,
  recommendNextActivity,

  // And 100+ more exports...
} from "mama-montessori";
```

### Raspberry Pi Deployment

```bash
# On a Raspberry Pi 4 (4GB+)
git clone https://github.com/OliWoods-Org/mama-montessori.git
cd mama-montessori
npm install

# Download offline content pack
npx mama-montessori download-pack core-3-6
npx mama-montessori download-pack llm-tiny

# Start offline server
npx mama-montessori serve --offline --port 3000

# Access from any device on the local network
# http://<pi-ip>:3000
```

**Total hardware cost: ~$50 (Raspberry Pi 4) + free software = complete Montessori AI education system.**

Compare that to $15,000-$30,000/year for a Montessori school.

## The Montessori Method in AI

### What we keep:
- **Follow the child** -- the AI observes, then presents. It never forces.
- **Prepared environment** -- digital materials organized for independent exploration
- **Three-period lesson** -- naming, recognition, recall (the proven learning sequence)
- **Sensitive periods** -- recognize when a child is ready for specific learning
- **Mixed-age collaboration** -- peer learning features, older helping younger
- **Practical life** -- real-world skills, not just academic content
- **Cosmic education** -- everything connects, the child finds their place
- **Control of error** -- materials are self-correcting, building independence
- **Normalization** -- tracking and supporting the development of sustained concentration

### What we do NOT do:
- No screen addiction mechanics (timers, limits, breaks built in)
- No replacing human teachers or parents -- we supplement
- No standardized testing pressure -- mastery-based, self-paced
- No data monetization -- you are not the product
- No walled garden -- all content is open, forkable, improvable
- No competitive gamification -- no leaderboards, no rankings
- No surveillance of children or families

## Safety & Compliance

- **COPPA 2025 compliant** -- parental consent required for children under 13
- **FERPA compliant** -- student data protected
- **GDPR-Children compliant** -- EU child data protections
- **LEAP risk assessment** -- 7-domain wellbeing monitoring (emotional regulation, social connection, academic engagement, self-efficacy, dependency risk, screen balance, family context)
- **No student data sold. Ever.** Period.
- **No advertising** to children. Zero.
- **No engagement manipulation** -- intrinsic motivation only
- **Crisis protocol** -- automatic escalation for safety concerns
- **Age-appropriate content guardrails** at every level

## Curriculum Coverage

### Ages 3-6 (Primary / First Plane)
- **Practical Life:** Pouring, spooning, folding, food preparation, care of self and environment
- **Sensorial:** Pink Tower, Brown Stair, Color Tablets, Sound Cylinders, Binomial Cube
- **Language:** Sandpaper Letters, Moveable Alphabet, phonetic reading, early writing
- **Mathematics:** Number Rods, Golden Beads, Spindle Box, decimal system
- **Cultural:** Continent Globe, puzzle maps, nature walks, music, art

### Ages 6-12 (Elementary / Second Plane)
- **Great Lessons:** Origin of the Universe, Coming of Life, Coming of Humans, Story of Language, Story of Numbers
- **Language Arts:** Grammar with grammar symbols, composition, research skills, literature
- **Mathematics:** Stamp Game, Checkerboard, fractions, geometry, pre-algebra
- **Science:** Botany, zoology, physics, chemistry, earth science
- **History & Geography:** Timeline of Life, civilizations, map-making

### Ages 12-18 (Adolescent / Third Plane)
- **Erdkinder concept** -- real-world application of knowledge
- **Micro-enterprise projects** -- business planning, financial literacy
- **Community service and social justice**
- **Advanced academics** -- preparation for any path
- **Identity development and philosophical exploration**
- **Career exploration and earning pathways**
